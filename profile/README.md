## Шерлок ТГ — мощный бот для пробива в Телеграм

**Шерлок бот ТГ** — это современный Telegram-бот для быстрого и точного поиска информации о людях. Пробив через Шерлок в ТГ занимает считанные секунды и выдаёт максимально полные данные из открытых источников.

### Что умеет Шерлок бот в Телеграм

- 🔎 **Пробив по номеру телефона** — узнайте владельца за секунды
- 👤 **Поиск по имени и фамилии** — находите профили и связанные данные
- 🚗 **Пробив по авто** — информация по госномеру
- 📱 **Поиск по Telegram** — данные по username и ID
- 🌐 **Поиск по соцсетям** — агрегация из открытых источников

## Как начать пользоваться

1. Перейдите в [Шерлок Бот](https://serlock-bot.org/)
2. Нажмите «Старт»
3. Введите номер телефона, имя или username
4. Получите результат мгновенно

## Бесплатный Шерлок в ТГ — начни без вложений

Ищете **бесплатный Шерлок в ТГ**? При первом запуске бота вы получаете бесплатные запросы для тестирования всех функций. Оцените качество пробива через Шерлок бот ТГ бесплатно — без подписок и скрытых платежей на старте.

## Шерлок ТГ Пробив — как это работает

**Шерлок ТГ пробив** использует агрегацию данных из открытых источников (OSINT). Шерлок бот ТГ пробив выполняет поиск по:

- Утечкам баз данных
- Открытым профилям социальных сетей
- Публичным реестрам
- Telegram-аккаунтам и группам

Все данные собираются исключительно из легальных открытых источников.
