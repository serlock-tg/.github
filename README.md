# 🔍 Шерлок ТГ — Telegram бот для пробива 2026

[![Шерлок Бот](https://img.shields.io/badge/Telegram-Шерлок%20Бот-blue?style=for-the-badge&logo=telegram)](https://serlock-bot.org/)

**Шерлок ТГ** — бот для пробива в Телеграм. Поиск информации по номеру телефона, username, нику, ФИО. Аналог Глаз Бога с расширенными базами.
